深拷贝、浅拷贝：

浅拷贝：用object.assign或 ... 展开符。但浅拷贝只解决了第一层的问题。

```javascript
//object.assign方式
let a = {
age:1,
color : {
    backgroundcolor:"green"
};
let b = object.assign({},a);
a.age = 2;
a.color.backgroundcolor = "red"
console.log(b.age); //1
console.log(b.color.backgroundcolor);//red
// ...展开符方法
let a = {
age:1,
color : {
    backgroundcolor:green
};
let b = {...a};
a.age = 2;
a.color.backgroundcolor = "red"
console.log(b.age); //1
console.log(b.color.backgroundcolor);//red
```
